# 描述


- vue单页应用，前后端分离
- 移动设备兼容：使用flexible.js和rem处理兼容问题
- 路由懒加载：Vue Router 处理路由
- axios做ajax请求
- vue-awesome-swiper 触摸滑动组件完成导航组件
- mint-UI完成图片懒加载、下拉刷新、infinite-scroll、navbar等组件
- 图标引入 import Icon from 'vue-awesome/components/Icon'
- esay-mock模拟数据
- stylus编写样式
